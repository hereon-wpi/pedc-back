
# Pre Experiment Data Collector (backend)

Uses [Nest](https://github.com/nestjs/nest) framework.

# Requirements

Need running mongodb on port 27017 with "pedc" database.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
